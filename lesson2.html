LESSON 2:

Information Systems


Information Systems (IS) integrates technology, people, and processes to collect, process, and manage data, enabling organizations to make informed decisions and improve efficiency. It bridges the gap between business needs and technological solutions, encompassing areas such as databases, software development, and system analysis to drive innovation and organizational success.

Data vs. Information

Data

- is a collection of raw, unorganized facts and details like text, observations,
figures, symbols, and descriptions of things
- no significance by itself
- measured in terms of bits and bytes
(1 byte = 8 bits)

1. Quantitative data - measures of values
2. Qualitative data - description of characteristics

Information

- processed, organized, interpreted, and structured data
- ex. graphs, reports
- provide valuable insights to guide effective decision-making


Components of Information Systems

Computer Hardware

Computers (allow access to the server) and servers (provide storage space for data, programs, and applications)

Computer Software

I. System Software

- Runs a computer’s internal functioning and peripherals
- Uses system programming languages

> Operating systems
    - principal system software
    - ex. Windows, MacOS, and Linux
    - manages a computer’s resources for the user’s utilization via GUI (graphical user interface)

II. Application Software

- Runs on top of system software
- Processes data for a user
- Use general-purpose languages

Networks / Telecommunications

- Used to connect computer systems from anywhere

Examples:
1. LANs (local area networks) - collection of devices connected together in one physical location or one building
2. WANs (wide area networks) - connects devices in multiple locations
3. Intranets - private network for sharing company information and computing resources among employees or within the company
4. Cloud networks - allow users to access and manage data, applications, and other resources remotely over the internet

A. Wired

- Coaxial cable - self-shielded cable for transmission of communication signals
- Fiber optics - transmits data, voice, and images by a passage of light through thin, transparent (plastic or glass) fibers.

B. Wireless

- Transmission of microwaves and radio waves
- Ex. RFID (radio frequency identification devices) sensor attached to products

Database

Is a collection of interrelated data organized so that records can be retrieved later

Human Resources

- Administrators who manage the system, configure hardware and software, and troubleshoot issues
- Users who learnt to perform tasks within the system to get work done

Procedures

- For using, operating, and maintaining an information system
- From setting up secure user accounts to creating emergency backup plans


Basic Processes

1. Input - from sensors, keyboards, scanners, or databases
2. Processing - from sensors, keyboards, scanners, or databases
3. Storage - in a database, file system, or cloud storage
4. Output - in an easy-to-understand format, such as reports, graphs, charts, or dashboards
5. Feedback - assessment from users and other stakeholders to evaluate the performance and improve the design and functionality


Supporting Systems in Information System

I. Executive Information System

- for executives
- provides comprehensive information from internal and external sources, enabling leaders to analyze trends, assess organizational performance, and formulate
long-term strategies

II. Decision Support System

- for senior managers who process information to provide insights to enable managers to navigate uncertainties, explore alternatives, and make well-informed
decisions
- harnessing advanced analytics, data modeling, and artificial intelligence to provide insights that can improve the company’s processes
- ex. comparing sales data between two different time periods to predict trends

III. Management Information System

- for middle managers who merge, sort, and organize data from TPS
- bridges gap between raw data and strategic decision-making within an organization

IV. Transaction Processing System

- for workers that provide key data
- processes, records, and monitors transactions in real time
