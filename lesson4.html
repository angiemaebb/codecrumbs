LESSON 4:

Computer System


A computer system is an integrated set of hardware and software components designed to perform a variety of tasks, from processing data to executing complex applications. It typically includes input devices, processing units, output devices, and storage devices. Together, these elements allow users to input, process, store, and output information, making it a versatile tool in various fields such as business, education, and science.

Computer Hardware

Computer hardware are the physical, tangible parts of a computer that can be touched. It includes primary electronic components essential for the
computer’s function. Examples include the processor, memory devices, monitor, printer, keyboard, mouse, and CPU.

Input Devices

allow users to enter data


Mouse
- hand-held pointing device for moving the cursor and interacting with on-screen object
- can be wired or wireless.
                            

Keyboard
- has QWERTY layout, allowing entry of text numbers, and special characters
                            

Microphone
- device that converts sound waves into electrical signals for audio input or communication
                            

Webcam
- captures visual images and streams them to the computer or network
                            

Touchpad
- built-in or external input device for controlling the on-screen pointer, often used on laptops as an alternative to a mouse

Output Devices

display the results of tasks


Monitor
- screen that displays information, images, and documents from the computer
                            

Printer
- converts electronic data into printed material
                            

Speakers
- external audio device that produces sound from the computer
                            

Headphones / Earphones
- personal audio devices that deliver sound output to a single listener
                            

Projector
- projects computer screen images into a larger surface for group viewing

Secondary Storage

store data for future use


1.Fixed Storage
- internal storage not easily removed without technical assistance
- internal flash memory, SSD, HDD

2.Removable Storage
- external, portable storage that can be connected/disconnected as needed
- optical discs, memory cards, floppy disks, magnetics tapes

Common secondary storage devices:
> Floppy Disc
> Compact Disc (CD)
    - CD-R (write once)
    - CD-RW (rewritable)
> Digital Versatile Disc (DVD)
    - DVD-ROM
    - DVD-R
    - DVD-RW
> Blu-ray Disc
> Hard Disk Drive (HDD)
> Flash Drive
> Solid-State Drive (SSD)
> SD Card

Internal Components

essential internal devices


Motherboard
- central circuit board that connects and supports the CPU and other internal hardware components
                            

(CPU) Central Processing Unit
- the “brain” of the computer, responsible for processing and executing instructions. It’s speed impacts overall performance
                            

(RAM) Random Access Memory
- temporary, volatile memory that provides immediate data access to programs; data is cleared when power is off
                            

Hard Drive
- physical storage device for both permanent and temporary data, including OS, programs, and files
                            

(SSD) Sold-State Drive
- non-volatile storage using NAND flash memory; retains data even without power
                            

(ODD) Optical Disk Drivce
- uses a laser light to read data from or write data to an optical disc
                            

Heat Sink
- passive cooling components that absorbs and dissipates heat, often placed on top of the CPU
                            

(GPU) Graphics Processing Unit
- a chip that handles graphical data processing, working alongside the CPU
                            

(NIC) Network Interface Card
- a board or chip that connects the computer to a network, commonly supporting Ethernet


Computer Software

A collection of instructions and procedures that execute tasks on a computer. Software is code that runs on a processor,
either at machine level or for an operating system.

I. System Software

operates hardware and manages internal functions, including operating systems, language processors, and device drivers.


A. (OS) Operating System
- manages hardware resources and enables other software to function
- examples: Microsoft Windows, Linux, MacOS, Android

types of operating systems:
> Graphical User Interface (GUI) - uses icons and visual indicator for interaction
> Command-based OS - uses text-based commands to interact with the system
> Single-user OS - designed for one user at a time
> Multi-user OS - allows multiple users to access a computer at the same time
> Network OS - manages network resources and allows communication between connected computers

B. Utility Software
- performs maintenance and management tasks for computer systems
- examples: anti-virus software, archivers backup software, clipboard manager, cryptographic utilities, data compression software, disk cleaner,
system monitor, file manager, disk defragmenter

C. Language Translators
- converts source code into machine-readable code

types:
> Assemblers - converts assembly language to machine code
> Compilers - translate high-level code into machine code before execution
> Interpreters - translates code line-by-line during execution

D. Device Drivers
- a specialized program that allows the operating system to communicate with and control hardware devices connected to the computer

types of device drivers:
> printer drivers
> graphics card drivers
> audio drivers
> network drivers
> storage drivers
> USB drivers

II. Application Software

performs specific tasks for users, such as word processing or spreadsheets.


A. General-purpose applications
- widely used for tasks like word processing, spreadsheets, etc.
- examples: word processing software, desktop publishing, database software, graphics software, presentation software, web browsers

B. Specialist applications
- developed for specialized tasks or industries
examples: accounting software, payroll software, banking software, educational software, statistical software, hospital management
software, enterprise resource planning (ERP), customer relationship management (CRM), database (DB), project management software (PMS),
business process management (BPM), productivity software, resource management software (RMLS), time management software
